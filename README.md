# Express JS Boilerplate
An basic express boilerplate for APIs

## Requirements
 - Node  16.17 LTS

## Dependencies
- Express JS
- Dotenv
- prisma

## Files Structure
```
├── index.ts
├── package.json
├── prisma
│   └── schema.prisma
├── README.md
├── src
│   ├── controllers
│   │   └── example.controller.ts
│   ├── database
│   │   └── client.ts
│   ├── middlewares
│   ├── router.ts
│   ├── server.ts
│   └── services
├── tsconfig.json
└── yarn.lock
```
